# Superstore Sales Analysis

## About
This project analyses Superstore sales data using SQL Server and Tableau.

## Tools Used
- SQL Server Express / SSMS
- Tableau
- Python (for data loading)

## Files
- SQLQuery1.sql, SQLQuery2.sql, SQLQuery3.sql - SQL queries for analysis
- Book1 - Tableau workbook with visualisations
- Superstore - source dataset
- query1_discount_profit, query2_loss_by_region, query3_shipmode_profit - query results
- load_superstore.py - Python script to load data
